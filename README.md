# AI-Конвейер · Песочница

Интерактивная вкладка-песочница для курса по ИИ: реальные бизнесы центра Москвы без сайта на карте, разбор по каждому и генератор промпта для Lovable.

**Демо:** см. GitHub Pages этого репозитория.

Один файл `index.html`, без бэкенда. Карта — OpenStreetMap (Leaflet, тёмные тайлы CARTO заменены на светлые), ключи не нужны.
